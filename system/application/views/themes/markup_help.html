Feedvolley themes use a subset of Tumblr's markup, existing Tumblr themes can usually be used with little or no modifications. All Tumblr tags are supported - ie, will not show up in the rendered HTML - but only the following are relevant for Feedvolley's purposes (all tags are case sensitive):

 

Page Elements

    •   {Title} Page title - either user customized or, by default, RSS feed's <title> tag
    •   {RSS} Feed URL
    •   {Description} Feed description (RSS <description> tag)
    •   {Favicon} Feed favicon URL

 

Item Elements

  •   {block:Text} or {block:Regular} Limit a single item (post) block (RSS <item> tag)
    •   {Permalink}Item URL (RSS <link> tag)
    •   {Title} Item title (RSS <title> tag)
    •   {Description} Content of RSS item's <description> tag
    •   {Enclosure} Content of RSS item's <enclosure> tag
    •   {Body} {Description} + {Enclosure} - will omit the enclosure if its URL is already present in description

 

Dates

  •   {block:SameDayDate} and {block:NewDayDate} let you distinguish between posts published on different days.
  •    All Tumblr's date elements are supported, including {DayOfMonth}, {DayOfMonthWithZero}, {DayOfWeek}, {ShortDayOfWeek}, {DayOfWeekNumber}, {DayOfMonthSuffix}, {DayOfYear}, {WeekOfYear}, {Month}, {ShortMonth}, {MonthNumber}, {MonthNumberWithZero}, {Year}, {ShortYear}, {AmPm}, {CapitalAmPm}, {12Hour}, {24Hour}, {12HourWithZero}, {24HourWithZero}, {Minutes}, {Seconds}, {Beats}

 

Custom Colors

  •    You can allow easy color customization by using a meta name tag to define a color name and code. Feedvolley will place the color code instead of that color name in the HTML or CSS. For example: <meta name="color:Background" content="#eee"/> lets you use <body bgcolor="{color:Background}">
